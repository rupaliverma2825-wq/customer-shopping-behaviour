# customer-shopping-behaviour

1) Customer Shopping Behaviour-

  Customer shopping behavior analysis examines how customers interact with products, pricing, promotions, and channels before purchasing. This project identifies     patterns in purchase preferences, buying frequency, demand across product categories, seasonal trends, and demographic influences. Understanding these behaviors    enables businesses to improve sales forecasting, design targeted marketing strategies, personalize customer experiences, optimize inventory, and enhance overall    customer satisfaction.
